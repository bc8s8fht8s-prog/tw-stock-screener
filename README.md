# TW Stock Screener

## 專案目標

建立一套可於 iPhone 使用的台股自動選股系統。

## Version 1.0

每天交易日下午 15:10 自動執行。

掃描範圍：

- 上市普通股
- 上櫃普通股
- KY 普通股

排除：

- ETF
- ETN
- 權證
- 特別股
- 興櫃

## 選股條件

### 條件一

今日收盤價 > 昨日最高價（不能等於）

### 條件二

今日 OSC >= 昨日 OSC

MACD：

- DIF = 12
- EMA = 26
- MACD = 9

計算方式需與 XQ 一致。

## 顯示內容

依股票代號排序。

每筆顯示：

- 股票代號
- 股票名稱
- 收盤價
- 昨日最高價
- 今日 OSC
- 昨日 OSC

頁首顯示：

- 資料日期
- 更新時間
- 符合條件股票數

## 執行方式

GitHub Actions 每天下午 15:10 自動執行。

iPhone 直接查看結果。
